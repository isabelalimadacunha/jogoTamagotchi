Título do Projeto: Jogo Tamagotchi
Propósito do Projeto: Criar um jogo do Tamagotchi em Java usando o teclado.
Data: 10/05/2023
Como Começar: Criar a classe Tamagotchi e determinar seus atributos dentro do contrutor, métodos de acesso, métodos que façam o Tamagotchi realizar algo e condições para que o jogo termine. Depois criar a classe Principal que possui o método main.
Observações: Para que cada ação do Tamagotchi seja aleatória é utilizado a função Math.random() nos métodos que o Tamagotchi fica com sono, fica com fome ou fica entediado.
